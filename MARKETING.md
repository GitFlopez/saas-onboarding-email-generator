### Key Differentiators
1. **Behavioral branching built in** — activated vs. stuck user paths; no other template system includes this
2. **In-app message variants** — tooltip + modal copy for Intercom/Appcues/Pendo alongside emails
3. **Break-up email closer** — recovers 10-15% of churned accounts; most resources skip this
4. **Moment of value framework** — forces the user to define their "aha moment" before writing a single email
5. **Three-segment churn rescue** — at-risk / churned / win-back are distinct sequences, not one generic re-engagement
